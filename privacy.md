# 🔐 Política de Privacidade – Nexstar

Esta Política de Privacidade descreve como o bot Nexstar coleta, usa e protege informações dos usuários na plataforma Discord.

1. Ao utilizar o Nexstar, você concorda com esta Política de Privacidade.
2. O bot pode coletar informações básicas necessárias para seu funcionamento.
3. Os dados coletados podem incluir:
   - ID de usuários
   - ID de servidores
   - ID de canais
   - Comandos utilizados
4. O Nexstar não coleta dados sensíveis, como senhas ou informações pessoais privadas.
5. As informações coletadas são utilizadas apenas para:
   - Executar comandos
   - Melhorar funcionalidades
   - Garantir o funcionamento correto do bot
6. Nenhum dado é vendido, compartilhado ou distribuído para terceiros.
7. Os dados são armazenados de forma segura e protegida contra acessos não autorizados.
8. O usuário pode solicitar a exclusão de seus dados a qualquer momento.
9. Algumas informações podem ser armazenadas temporariamente para desempenho do bot.
10. O Nexstar não se responsabiliza por dados compartilhados voluntariamente por usuários em servidores.
11. Administradores de servidores são responsáveis pelas permissões concedidas ao bot.
12. O bot pode utilizar APIs externas para funcionamento de algumas funções.
13. Essas APIs podem ter suas próprias políticas de privacidade.
14. O Nexstar não tem controle sobre serviços externos utilizados.
15. Medidas de segurança são aplicadas para proteger os dados coletados.
16. Mesmo assim, nenhum sistema é 100% seguro.
17. O uso do bot é por conta e risco do usuário.
18. Esta política pode ser atualizada a qualquer momento.
19. Alterações entram em vigor imediatamente após publicação.
20. O uso contínuo do bot implica aceitação das mudanças.
21. O Nexstar não possui vínculo oficial com o Discord.
22. Para dúvidas ou solicitações, entre em contato com o desenvolvedor.
23. Recomendamos revisar esta política regularmente.
24. Esta política entra em vigor a partir do uso do bot.
25. Ao utilizar o Nexstar, você concorda com esta Política de Privacidade.
